## typescript中interface和type有什么区别？
|           | type         | interface                         |
|-----------|--------------|-----------------------------------|
| extend    | V2.7 later   | true                              |
| implement | V2.7 later   | true                              |
| type check| yes          | yes                               |
| new name  | don't create | create and can be used everywhere |
