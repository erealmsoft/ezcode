## static getDerivedStateFromError()和componentDidCatch()有什么区别？

### 对比
|类别|getDerivedStateFromError|componentDidCatch|
|----|------------------------|------------------|
|调用时间|渲染阶段调用|会在“提交”阶段被调用|
|执行其它操作|不可以|可以|
|参数|err|err,info|

## React 中 Element 与 Component 的区别？
-   一个是元素，一个是组件
-   组件：可复用的小的代码片段
-   元素：元素描述了你在屏幕上想看到的内容。React 元素是不可变对象
