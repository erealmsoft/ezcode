### 1.为什么要使用Typescript？
1. TS 是 JS的强类型版本，编译时去掉类型和语法，生成纯的 JS 代码。
2. TS 是 JS的超集， 支持所有的JS 语法，并在基础上进行扩展，如： class/ interface / moudle 
3. TS 在浏览器中最终运行的仍旧是 JS， 所以TS 不依赖于浏览器，无兼容性问题。
4. 优点： 静态类型检查， 代码重构， 可读性强。
### 2.readonly和const有什么区别？
1. const
- 用于变量；
- 变量不能重新赋值给其他任何事物。
2. readonly 
- 用于属性；
- 用于别名，可以修改属性；
